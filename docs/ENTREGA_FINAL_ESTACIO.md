## 1. Visão Geral do Projeto
O projeto **CineGestão** consiste em uma solução mobile desenvolvida para automatizar o gerenciamento de locadoras de filmes independentes. O foco principal é a transição de processos manuais para uma plataforma digital acessível, permitindo o controle de acervo, clientes e fluxos de locação em tempo real.

## 2. Tecnologias Utilizadas
- **Frontend**: React Native com TypeScript (Interface Moderna e Dark Mode).
- **Backend**: Node.js com Express (API RESTful).
- **Banco de Dados**: MongoDB (Persistência de Dados NoSQL).
- **Controle de Versão**: Git e GitHub.
