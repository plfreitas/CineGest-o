# Relatório de Visita Técnica - CineGestão

**Data da Visita**: 29/04/2026
**Local**: Freitas Vídeo Locadora - Centro, Rio de Janeiro.
**Responsável Técnico**: Pedro Luis Santos Freitas
**Parceiro**: Sr. Pedro Freitas (Proprietário)

## 1. Objetivo da Visita
Coleta de requisitos finais e validação do fluxo de trabalho manual atual para transição para o sistema digital CineGestão.

## 2. Atividades Realizadas
- Observação do processo de locação física (registro em caderno/fichas).
- Identificação de gargalos: demora na consulta de disponibilidade e dificuldade em rastrear devoluções atrasadas.
- Apresentação do protótipo inicial do CineGestão para feedback do proprietário.

## 3. Resultados e Feedbacks
- O parceiro validou a necessidade de um status visual claro (Verde/Vermelho) para filmes.
- Foi solicitada a inclusão de um campo de CPF para garantir a segurança nas locações.
- A interface dark-mode foi elogiada pela legibilidade em ambientes de baixa luz (comuns em locadoras).

## 4. Próximos Passos
- Implementação do banco de dados MongoDB para persistência.
- Teste de usabilidade com a primeira versão do APK (Android).
